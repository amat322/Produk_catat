# 📦 Amat Bajualan - Manajemen Stok

Aplikasi web sederhana untuk membantu pengelolaan stok produk dan pembuatan katalog WhatsApp secara instan. Dibuat khusus untuk mendukung kemudahan operasional **Amat Bajualan**.

## 🚀 Fitur Unggulan
- **Branding Personal**: Dilengkapi logo eksklusif "Amat Bajualan".
- **Manajemen Produk**: Tambah, edit, dan cari produk dengan harga serta sisa stok.
- **Status Stok Otomatis**: Label **HABIS** otomatis muncul jika stok 0.
- **WhatsApp Integration**: Kirim katalog rapi langsung ke pelanggan via WA.
- **Data Aman**: Menggunakan penyimpanan lokal browser (Data tidak hilang saat refresh).
- **Excel Support**: Bisa impor data dari Excel atau ekspor laporan stok ke file `.xlsx`.

## 🛠️ Cara Penggunaan
1. **Input**: Masukkan detail produk di form atas.
2. **Cari**: Gunakan kolom pencarian untuk mengecek stok dengan cepat.
3. **Share**: Klik tombol WhatsApp untuk promosi ke pelanggan.
4. **Backup**: Ekspor ke Excel secara berkala untuk cadangan data.
